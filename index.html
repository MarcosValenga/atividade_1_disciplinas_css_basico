ADS - 2024

O curso de Análise e Desenvolvimento de Sistemas (ADS) é uma formação superior que visa capacitar os alunos para atuar na criação, desenvolvimento e manutenção de sistemas computacionais. Durante o curso, os estudantes aprendem a projetar e implementar soluções tecnológicas que atendem às necessidades de empresas e organizações.


Disciplinas

  • Desenvolvimento de aplicações para internet

    Estudo de linguagem de programação para desenvolvimento de aplicações voltados para Internet. Desenvolvimento de aplicações Web cliente/servidor. Tecnologias Web. HTML. Linguagens de programação para Web. Arquitetura de aplicações Web. Usabilidade, tendências e frameworks para desenvolvimento Web.

    Prof: Deivid Sala: 118

  • Atividades Práticas Interdisciplinares de Extensão

    Criação de Projetos envolvendo o curso.

    Prof: Anderson Sala: 113

  • Análise e Modelagem de Sistemas

    Estudo de Análise para criação de sistemas, envolvendo documentação, levantamento de requisitos, criação de diagramas UML, e bla bla bla...

    Prof: Rafael Sala: 118

  • Front-End Frameworks

    Estudos envolvendo métodos predefinos para montagem de sites, de forma facilitada, ou seja, em componentes. Os alguns dos principais frameworks Front-End são Angular, React, Vue e Flutter(esse é bom pode usar 👍).

    Prof: Deivid Sala: 118

  • Banco de Dados

    Um banco de dados é uma coleção organizada de informações - ou dados - estruturadas, normalmente armazenadas eletronicamente em um sistema de computador.

    Prof: Rafael Sala: 118

  • Programação Orientada a Objetos e Estrutura de Dados

    A programação orientada a objetos (POO) é um paradigma de programação que se baseia no conceito de “objetos”. Esses objetos são entidades que possuem atributos, que são valores associados a eles, e métodos, que são ações que os objetos podem realizar. A ideia por trás da programação orientada a objetos é que os objetos podem ser usados para representar elementos do mundo real de maneira mais precisa e flexível, tornando mais fácil criar programas que sejam extensíveis, reutilizáveis e manuteníveis.

    Prof: Rafael Sala: 118
